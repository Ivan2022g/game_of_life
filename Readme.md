# Анализ условий игры «Game of Life»

## Входные данные
- Начальное состояние поля размером **80×25** клеток подаётся через **стандартный ввод (stdin)**.
- Предоставляется **не менее 5 файлов** с готовыми начальными конфигурациями, которые можно загрузить через перенаправление ввода (например, `./game_of_life < glider.txt`).
- **Формат входных данных (stdin или файла):**
  - Каждая строка содержит координаты **одной живой клетки** — два целых числа, разделённых пробелом: `x y`
  - **Начало координат** — левый верхний угол поля.
  - **Ось X** направлена **вправо** (номер столбца, от 0 до 79).
  - **Ось Y** направлена **вниз** (номер строки, от 0 до 24).
  - Все клетки, не перечисленные во входных данных, считаются **мёртвыми**.
  - Пример строки файла: `10 5` (клетка на 11-м столбце в 6-й строке становится живой).

## Условия окончания игры
- Игра завершается **только по инициативе пользователя** — нажатием клавиши **`Space Bar`** (пробел).
- Автоматическое завершение (например, при вымирании всех клеток или достижении стабильной конфигурации) **не предусмотрено** условиями задачи.

## Правила
- **Игровое поле** — прямоугольник 80×25 клеток, **замкнутый сам на себя** (тороидальная топология):
  - Правый край соединён с левым.
  - Нижний край соединён с верхним.
- **Состояния клеток**: каждая клетка может быть живой (заполненной) или мёртвой (пустой).
- **Эволюция** (одно поколение) вычисляется одновременно для всех клеток по классическим правилам Джона Конвея:
  1. **Выживание**: живая клетка с **2 или 3** живыми соседями остаётся живой.
  2. **Смерть от одиночества**: живая клетка с **менее чем 2** живыми соседями умирает.
  3. **Смерть от перенаселённости**: живая клетка с **более чем 3** живыми соседями умирает.
  4. **Рождение**: мёртвая клетка с **ровно 3** живыми соседями становится живой.
- **Соседи**: клетка окружена **восемью** клетками (по горизонтали, вертикали и диагонали), определяемыми с учётом замыкания поля.

## Дополнительные параметры (управление скоростью)
- **`A`** — увеличение скорости смены поколений.
- **`Z`** — уменьшение скорости смены поколений.
